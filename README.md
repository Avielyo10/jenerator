# Jenkins Plugin Generator

## Usage
<pre>
jenkinsPluginGen --name (toolName) [options ...]  
--name       :the tool name  
-A           :make it all - used by default  
-B           :make Builder  
-I           :make Installation  
-P           :make Publisher  
-U           :make Util  
-W           :make Wrapper  
-h, --help   :call help and exit  
</p>
